Starting Instructions

1. run `cd streamlit`
2. Create the Python Enviroment
    1. Run `python3 -m venv venv`
    2. Run `pip install -r requirements.txt`
3. Run the app
    1. Run `source venv/bin/activate`
    2. Start the app with `streamlit run app.py`
